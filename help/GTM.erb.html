To use GTM:

  • Open(or create) your project in XCode
  • Create a new target for running unit tests:
    • Right click on your application target and choose 'Duplicate'
    • Right click on the new target and select 'Get Info'
    • Rename the newly created target to (EXISTING_TARGET)UnitTest
    • On the 'Build' tab search for 'Product Name' and set the product name to '(EXISTING_PRODUCT)UnitTest'
    • Delete 'main.m' from the Test targets's 'Compile Sources''
  • Create a new group to for your test dependencies
    • Right-click on your project in the 'Groups and Files' sidebar
    • Click 'Add>New Group'
    • Create a top level group 'Vendor'
    • Under 'Vendor' group, create another group called 'Test'
    • Right click on the 'Test' group and choose 'Add>Existing Files...'
    • When the file dialog pops up, choose the GTM directory
    • Add the newly created group to the UnitTest target
  • Wire everything up together
  • Add a new build phase to run our tests
  • Set the script to 'sh Vendor/Test/GTM/UnitTesting/RunIPhoneUnitTest.sh'
  • Add the unit tests
  • Create a new Group called 'Tests'
  • Add existing files from the 'Tests/Unit' directory of your project
  • Select the correct target and click 'Build and Run' to launch the tests
